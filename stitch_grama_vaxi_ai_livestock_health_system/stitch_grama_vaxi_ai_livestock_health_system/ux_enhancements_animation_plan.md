# Grama-Vaxi UX Enhancements & Animations

## New Features
1. **Live Camp Locator (Map-centric):** Interactive map showing nearby mobile vet clinics and vaccination camps with distance and status.
2. **Dynamic Registration Flow:** A multi-step form with progress indicators and micro-interactions for selecting animal types.

## Animation Strategy
1. **Lottie Micro-interactions:** Use subtle animations for success states (checkmarks) and empty states.
2. **Staggered Card Loading:** Dashboard items should slide in with a slight delay for a "premium" feel.
3. **Skeleton Loading:** Shimmer effects for offline-first data fetching.
4. **Active Navigation:** The bottom bar icon should 'pop' and expand when selected.
5. **Progress Bars:** Smooth linear interpolation for the vaccination progress on the Health Card.
